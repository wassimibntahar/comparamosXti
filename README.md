# 📱 **ComparamosXTi**

## ✨ **Sobre el proyecto**
**ComparamosXTi** es una plataforma web diseñada para facilitar la vida tanto a compradores como vendedores de productos tecnológicos en el mercado de segunda mano: smartphones, consolas, tablets y más.

Como persona que se dedica día a día a la **compra-venta online**, recibo decenas de mensajes de potenciales clientes con las mismas dudas:

- “¿Cuánto vale mi móvil ahora?”  
- “¿Cuánto pagas por este modelo?”  
- “¿Cuál es el precio justo?”  
- “No tengo ni idea si estoy pagando de más o vendiendo demasiado barato.”

Tanto vendedores como compradores llegan **perdidos**, sin entender el valor real del dispositivo.  
Muchos corren el riesgo de **malvender lo suyo**, y otros de **pagar demasiado** o caer en estafas por falta de información.

Este proyecto nace como una solución de valor para todos ellos y como una forma de **optimizar mi tiempo**, evitando responder las mismas preguntas una y otra vez.

---

## 🎯 **Objetivo de la web**
El objetivo principal es proporcionar una herramienta clara, accesible y rápida para que cualquier usuario pueda:

- Consultar el **precio real y actualizado** de un dispositivo.  
- Comparar modelos de forma inmediata.  
- Ver **características esenciales** sin perder tiempo buscando.  
- Evaluar si vale la pena comprar o vender ese dispositivo.  
- Obtener una referencia fiable para evitar **engaños, precios injustos o estafas**.  

---

## 🧩 **¿Qué problemas resuelve?**

### ✔ Evita la incertidumbre del comprador/vendedor  
Ya no es necesario buscar en mil páginas, preguntar a desconocidos o arriesgarse a una mala operación.

### ✔ Ahorra muchísimo tiempo  
El usuario obtiene una valoración al instante.  
Yo ahorro mensajes repetitivos, explicaciones y negociaciones innecesarias.

### ✔ Información clara, simple y visual  
Todo está organizado de forma intuitiva en categorías:

- **Móviles**  
- **Consolas**  
- **Ordenadores** (próximamente)  
- Y más dispositivos pronto

### ✔ Adaptado al mercado real  
Los precios no son “inventos aleatorios”, sino valores basados en **experiencia real de compra-venta** y en el movimiento actual del mercado.

### ✔ Ayuda a evitar estafas  
Muchos compradores desconocen los precios reales y terminan pagando más de lo que vale el producto.  
La web actúa como **guía actualizada** para ayudarles a tomar decisiones seguras.

## 🛠️ **Tecnologías utilizadas**

- **HTML5** — estructura del sitio  
- **CSS3** — estilos y diseño  
- **JavaScript** — lógica e interactividad  
- **Fetch API** — carga de datos  
- **JSON** — base de datos simple  
- **Hostinger** — hosting  
- **FileZilla (FTP)** — gestión de archivos  
- **Responsive Design** — optimizado para móviles  

El sitio es **estático, rápido y ligero** para garantizar una experiencia fluida.

## ❤️ **Motivación personal**
Este proyecto nació por una necesidad real:  
en la compra-venta, la mayoría de personas **no conocen el verdadero valor** de un dispositivo.  
Muchos vendedores terminan perdiendo dinero, y muchos compradores pagan más de lo necesario… o incluso caen en estafas.

**ComparamosXTi** convierte ese problema en:

- una herramienta útil,  
- una experiencia más profesional,  
- una plataforma transparente,  
- y un punto de referencia fiable para cualquier usuario del mercado de segunda mano.
