# 📱 **ComparamosXTi**

## ✨ **Sobre el proyecto**
**ComparamosXTi** es una plataforma web creada para facilitar la vida a los usuarios que quieren comprar o vender **smartphones, consolas y otros dispositivos electrónicos** en el mercado de segunda mano.

Como persona que se dedica día a día a la **compra-venta online**, recibo decenas de mensajes de potenciales clientes con las mismas dudas:

- “¿Cuánto vale mi móvil ahora?”
- “¿Cuánto pagas por este modelo?”
- “¿Cuál es el precio más justo?”
- “No tengo ni idea si lo estoy vendiendo caro o barato.”

La mayoría llegan **perdidos**, sin entender el valor real de su dispositivo.  
Este proyecto nace como una solución de valor para ellos y una forma de **optimizar mi tiempo**, evitando responder manualmente las mismas preguntas una y otra vez.

---

## 🎯 **Objetivo de la web**
Hacer que cualquier usuario pueda:

- Consultar el **precio real** de su dispositivo.  
- Comparar **múltiples modelos** de forma rápida.  
- Ver **características esenciales**.  
- Evaluar si vale la pena vender/comprar ese modelo.  
- Obtener una referencia inmediata, **sin depender de terceros**.

---

## 🧩 **¿Qué problemas resuelve?**

### ✔ Evita la incertidumbre del comprador/vendedor  
Ya no hace falta buscar en mil sitios o arriesgarse a una mala venta.

### ✔ Ahorra muchísimo tiempo  
El usuario encuentra el precio al instante.  
Tú ahorras mensajes repetitivos y explicaciones constantes.

### ✔ Información clara, simple y visual  
Todo organizado por categorías:

- **Móviles**  
- **Consolas**  
- **Ordenadores (próximamente)**  
- Más dispositivos en camino

### ✔ Adaptado al mercado real  
No son precios inventados. Son valores basados en **experiencia real** de compra-venta.

---

## 🛠️ **Tecnologías utilizadas**

- **HTML5** — estructura del sitio  
- **CSS3** — diseño y estilos  
- **JavaScript** — interactividad  
- **Fetch API** — carga de datos  
- **JSON** — base de datos simple  
- **Hostinger** — hosting  
- **FileZilla (FTP)** — gestión de archivos  
- **Responsive Design** — optimizado para móviles  

El sitio es **estático, rápido y ligero**.

Este proyecto nació por **necesidad real**:  
en la compra-venta, la gente suele desconocer por completo el verdadero valor de su dispositivo, y la mitad del tiempo lo pasamos explicando precios.

**ComparamosXTi** convierte ese problema en:

- una herramienta útil,  
- una experiencia más profesional,  
- una plataforma que aporta valor REAL a los clientes.
